BirdLore.org

Help digitize historical bird records.

The process

Once this is set up the process will be:

  • Identify a historical record source
  • Verify it is not already documented on eBird
  • Transcribe to a digital format
  • Import to eBird

Criteria

Initially, all records should be either a published account of bird species observations, or a hand-written account by a known source.

Any textual data submitted should be accompanied by an original photocopy for verification.

Roadmap

This site obviously isn't ready yet to facilitate any of this. Some things that need to get done:

  • Create a login/user/registration system
  • Create a means of submitting a source (photo upload)
  • Create a way to browse sources
  • Add tools to transcribe sources to a format eBird can recognize (this is partly complete)
  • Add peer review process before submission to eBird
  • Allow for open data downloads of all data submitted here
